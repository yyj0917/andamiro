# Andamiro

하루 한줄 개인 일기 웹/앱 프로젝트입니다.

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- shadcn/ui
- Vitest
- Firebase

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm test
```

## Firebase

Firebase 콘솔에서 웹 앱을 만든 뒤 `.env.example`을 기준으로 `.env.local`을 채우면 됩니다.

Firestore는 `firestore.rules`를 Firebase 콘솔 Rules 탭에 붙여 넣거나 Firebase CLI로 배포해야 합니다.
